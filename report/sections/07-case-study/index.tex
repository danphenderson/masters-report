\section{Idealized Stenosis Case Study}
\label{sec:case-study}
\label{sec:idealized-stenosis-case-study}

This chapter uses one controlled stenosis computation to show how the
\texttt{StenosisHemodynamics} model record is specified, checked, and compared
through a declared observation operator. The case study fixes a smooth
idealized geometry, states the implemented one-dimensional area--flow model,
records the principal finite-volume realization, verifies the declared
operator, tests the geometry-rest state, and compares one-dimensional output
with available resolved three-dimensional velocity data.
